=== Настройка Open Server ===
При первом запуске заходим в Модули:
HTTP -  Apache_2.4-PHP_7.2-7.4
PHP - 7.4
MySQL - 8.0

Меню:
Убрать галочку с Открывать веб-стр в указанном браузере
Можно прописать путь к файлу VScode для второго  пункта
В меню основные можно поставить галочку на автозаупск и запуск вместе с виндой

При запуск сервера , он ищет файлики индекс и если у этого файла разширение PHP  то он отправляет файл в PHP Интерпритатор, дальше обрабатывет скрипты что там написанны серверу возвращает HTML и сервер в ответе дает HTML страницу
