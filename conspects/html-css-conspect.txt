=== Приоритеты у селекторов === 
Есть у не сколькоих теготов есть одинаковые классы и мы для них прописали какие значения, но хотим у конктрреного поменять а другие так и остались, нужно сделать селективный приоретет, например:
.text{
    font-size: 20px;
}
.classname p.text{
    font-size: 25px;
}
- Вложеность первого уровня
.classname>p{
    Обратимся только к тем <p></p> котроные находяться в классе на первом уровне(прямые дети класса)
}





Изображения
Если мы хотим утсановить какой то фон страницы или блока и он состоит из картинки которая могбы повторяться и не чего бы не менялось, вот такую брать и надо, вырезаем маленький квадрат , и она repeat и получаеться один большой фон
картинкам лучше прописать:
width: 100%; - а ширину контролировать у контейнера
object-fit: cover; - полняет пропорционально картинку по шырине контаинера
background-image: url(), url() - так можно указаывать несколько картинок, но увидим только первую, так как картинка повторяеться на всю ширину и высоту окна браузера и перекрывает вторую

background-color: #000; - Сначала надо указать просто цвет, для тех у кого не подрживает эти команды 
background-image: ridial-gradient() - градиент по окружности 
background-image: linear-gradient() - линейный
Крутые градиенты на этом сайте:
https://cssgradient.io/

background-clip: text - говорит на что  наложить background-image:
Станная тема но такое есть, рабоает в паре с background-clip:
text-fill-color: transparent; - говорит задать цвет текста прозрачным


background-repeat: no-repeat, repeat; - отменяет повторение первой а второй разрешает 
background-repeat:  repeat-x (repeat-y) - повторяет только по осям
background-size: 50px 200px;  Сначала идет ширина а потом высота 
background-attachment: fixed; - Фиксирует Изображения оно опускаеться вместе с скроллом
background-position: center  - станет по центру видимой части окна браузера.  top right - будет сверху справа. cover - растягивыаеться на всех браузер(контаинер ) пропорционально 



Отступы:
padding: 10px 20px: - Всегда начинаеться сверху, если два значение то сначала сверху-снизу а потом по бокам, если три тогда сверху слева-справа внизу
- по началу удобно прописать border что лучше видеть очератание контейнера и работать только с padding


Сылки:
Для body,html можно пременить команду которая делает плавный переход на внутринию сылку на стрице(якоря):
scroll-behaivor: smooth;
Что ты перейти к якорю на стрице нужно указать id елементу и в <a></a> прописать id:
<a href="#anchor">Anchor</a>


Формы:
<button type="reset">reset</button>
<input type="reset" value="reset">
Вот такие кнопки можно добавить в конце формы для очистки всех полей

В каждый идекст нужно добовлять атрибут tabindex="1"  и так далие для удобного перехада нажатием ТАБ
